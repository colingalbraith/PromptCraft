About PromptCraft

PromptCraft helps you create better prompts for AI systems like ChatGPT, Claude, or Gemini.

Simply enter your prompt, choose a style, and let PromptCraft enhance it!

Your API key is stored locally using Chrome's secure storage and is never sent to PromptCraft servers.

How to get a Gemini API Key:

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Create a new API key
  4. Copy and paste it here
